# AI Job Agent

This is an AI-powered job search tool that finds and ranks jobs automatically based on a user's role, skills, and location.

## What this project does
- Searches for jobs using AI
- Matches jobs with user skills
- Ranks jobs by relevance
- Shows job descriptions and apply links
- Saves time by automating job search

## Tech Used
- Python
- FastAPI
- React
- AI semantic matching

## How to run

### Backend
uvicorn backend.main:app --reload

### Frontend
cd frontend  
npm install  
npm run dev

## Why I built this
I built this project to learn how agent-based AI systems can automate real-world problems like job searching.
